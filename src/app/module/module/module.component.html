Module

Définition

ng generate module nom-du-module

Un module est une entité qui regroupe et organise des fonctionnalités liées dans une application. Techniquement, un module est une classe décorée avec @NgModule, qui sert à définir un contexte pour un ensemble de composants, directives, pipes, services, et autres modules.
- Grouper des fonctionnalité en blocs
- Organiser les parties de l'application
- Rassembler les composants et les réexporter dans d'autres modules

Structure

- declarations: lister les composants, les pipes etc qui appartiennent au module
- imports: importer d'autres modules utilisés dans celui-ci
- exports: les composants, les pipes, les directives qui sont utilisables lorsque le module est importé ailleurs
- providers: importer les services injectables
- bootstrap: le composant principal à charger
