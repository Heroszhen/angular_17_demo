Rxjs

Définition

RxJS (Reactive Extensions for JavaScript) est une bibliothèque clé dans Angular, utilisée pour la programmation réactive avec les observables.
Elle est largement intégrée dans Angular, notamment dans le système HttpClient, Forms, Router, et les services personnalisés.

API

- Observable
- Subscriber/Observer(next, error, complete)
- Subscribtion
- Subject(un object observable)
Subject, BehaviorSubject, ReplaySubject, AsyncSubject

CRUD

articles$ = new BehaviorSubject([]);
articles$.next(allArticles)

Create :
const articles = articles$.getValue();
articles$.next([...articles, oneArticle])


Update :
const articles = articles$.getValue();
articles[index] = newArticle;
articles$.next([...articles])

Delete :
const articles = articles$.getValue();
articles$.next(articles.filter(article => article.id !== id));
