Parent - Enfant

Je suis le parent


standalone, importer un composant

@Input, @Output

Le parent vers l'enfant

Dans parent ts, définir une valeur : title='My title'
Dans parent html, on met [title]='title' dans la balise de l'enfant
Dans enfant ts, on utilise le décorator Input pour recevoir la valeur de titre

Le parent utilise les infos de l'enfant

Dans parent html, on met #child dans la balise de l'enfant
Dans parent ts, on utilise le décorator ViewChild : @ViewChild(ChildComponent) child!: ChildComponent;
On peut utiliser les attributs et les methodes de l'enfant avec this.attribut, this.function

L'enfant vers le parent

Pour envoyer des infos de l'enfant vers le parent:
Dans l'enfant ts, on utilise le décorator Output et EventEmitter
@Output() messageEvent = new EventEmitter();
this.messageEvent.emit('value')
Dans le parent html, on met (messageEvent)="getMessageFromChild($event)" dans la balise de l'enfant
Dans le parent ts, on crée la fonction getMessageFromChild(e)
